= Add-Ons
  * Falcon add-ons and complementary packages.
    - https://github.com/falconry/falcon/wiki

  * falcon-rest
    - Auto REST API with with Falcon and SQLAlchemy
    - https://github.com/boomletsgo/falcon-rest

  * falcon-stats
    - A falcon middleware for gathering request-response statistics
    - https://github.com/dmuhs/falcon-stats

  * falcon-sqla
    - Falcon Middleware: SQLAlchemy Integration
    - https://github.com/vytas7/falcon-sqla

  * pytest-falcon-client
    - Pytest client fixture for the Falcon Framework.
    - https://github.com/sivakov512/pytest-falcon-client

  * falcon-jsonify
    - Falcon middleware to serialize/deserialize JSON with built-in request validator
    - https://github.com/AndreiRegiani/falcon-jsonify

  * falcom-marshmallow
    - Marshmallow serialization/deserialization middleware for Falcon
    - https://github.com/mplanchard/falcon-marshmallow

  * falcon-require-https
    - Falcon web framework middleware to sanity-check that HTTPS was used for the request.
    - https://github.com/falconry/falcon-require-https

  * falcon-autocrud
    - Makes RESTful CRUD easier
    - https://pypi.org/project/falcon-autocrud/
